- Makefile - это, файл, работающий с make-системой, которая позволяет автоматически запускать
   компилятор с нужными параметрами. 
   В рамках данного проекта, используемый компилятор будет - GCC, и в рамках данного компилятора
   команда сборки/запуска (к примеру) файла main.cpp будет выгл. след. обр.:
   gcc main.cpp
   Но! По мере разростания программые у нее появл. новые исходные файлы и т.д. Поэтому
   для удобство было решени использовать Makefile, чтобы автоматически собирать нашу программу.

- misc.cpp/misc.h - файлы в которых содержаться все вспомогательные (служебные) ф-ии нашего проекта.
